# 夸夸APP

描述：
1. AI 流式输出 + 粒子特效库集成 + DOM 转图片技术。
2. App首页中输入同事名字和优点，生成一段浮夸的赞美，并生成一张荣誉证书。

核心功能：
一. 核心功能 打字机夸夸
  1. 界面包含“被夸人”和“优点关键词”输入框。
  2. 点击“生成”后，模拟 AI 生成一段 100 字左右的浮夸赞美文案。
    - 要求：必须有打字机效果 (Typewriter Effect)，字要一个接一个地蹦出来，不能一次性显示。
二. 特效
  1. 要求让 Cursor 推荐并安装一个轻量级的粒子特效库。
  2. 交互逻辑：
    - 增加界面按钮，每次点击“点赞”按钮，屏幕上都要炸开五颜六色的纸屑。
    - 连续点击时，显示 x2, x3 ... x99。随着数字变大，数字的颜色要变红，且出现“抖动”动画（Shake Effect）
三. 荣誉证书
  1. 点击“颁发证书”按钮。
  2. DOM 转图片：利用 html2canvas 或类似库，将当前的赞美文案、被夸人名字、背景样式合成一张图片。
  3. 生成的图片可以下载。
